## 关于本站

本站使用 [Docsify](https://docsify.js.org/) 构建，无需构建步骤，直接渲染 Markdown 文件。

### 功能特点

- 📱 **移动端优化** - 适配手机阅读，字体大小、行高经过精心调整
- 🎨 **护眼配色** - 采用米黄色背景，减少视觉疲劳
- 🔍 **全文搜索** - 支持关键词搜索
- 📖 **章节导航** - 上一章/下一章快捷切换
- 📊 **阅读进度** - 顶部进度条显示阅读位置

2. 在 `_sidebar.md` 中添加章节链接
3. 提交并推送到 Gitee
